Special Instruction:

As I am using strcpy function for String copy, we need to add extra parameter in VS 2013 project setting. 
These are steps for that:
1. Right click o project and open properties.
2. Goto Configuration Properties -> C/C++
3. Edit "Preprocessor Definitions"
4. Add this line into it : "_CRT_SECURE_NO_WARNING"
5. Click apply and Ok.
